Basic events:-
	Mouse event:
		click (onclick)
		double click (ondblclick)
		right click (oncontextmenu)
		mouse hover (onmouseenter)
		mouse out (onmouseout)
		mouse down(onmousedown)
		mouse up (onmouseup)
	Windows event: (work only in body Tag)
		keypress (onkeypress)
		key up (onkeyup)
		page loading (onload)
		page unloading (onunload)
		page resize (onresize)
		scroll (onscroll)
