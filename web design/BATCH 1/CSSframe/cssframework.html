All CSS Grid properties:- Property Name :Description column-gap specifies the gap between the columns gap A shorthand property for the row-gap and column-gap property grid A shorthand property for the grid-template-rows ,grid-template-column grid-auto-rows, grid-auto-columns, and grid-auto-flow property grid-area Either specifies a name for the grid item or this property is a show start ,grid-column-start,grid-row-end, and grid-column-end property grid-auto-columns specifies a default column size grid-auto-flow Specifies how auto-placed item are inserted in the grid grid-auto-rows specifies a default row size grid-column A shorthand property for the grid-column-start and grid-column-end grid-column-end Specifies where the grid item should end in terms of columns grid-column-gap Specifies the size of the gap between column grid-column-start Specifies where the grid item should start in terms of columns grid-row A shorthand property for the grid-row-start and grid-row-end property grid-row-end Specifies where the grid item should end in terms of rows grid-row-gap Specifies the size of the gap between rows grid-row-start Specifies where the grid item should start in terms of rows grid-gap A shorthand property for the grid-row-gap and grid-column-gap property grid-template A shorthand property for the grid-template-rows and grid-template-column property grid-template-areas Specifies how to display column and rows ,using named grid item grid-template-columns Specifies the size of the column and how many column in a grid layout grid-template-rows Specifies the size of the rows and how many row in a grid layout row-gap Specifies the gap between the grid rows.
